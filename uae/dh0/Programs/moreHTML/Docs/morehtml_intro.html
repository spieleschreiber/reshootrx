Introduction

Use the tool "moreHTML" as you would use the tool "more", that means you can give the filename as an argument etc. You may also drop icons of HTML texts into the window to be displayed. Use it as the default viewer for HTML files if you want to have a fast loading viewer. Tables and frames are not supported, yet.

Be aware that you have to have the "RexxMast" running to use the toolbar. This toolbar is configurable. For better outlook "moreHTML" use the NewString.mcc (© Rüdiger Sopp) and ImageButton.mcc (© Maik Schreiber).
